
# Bookie

Bookie is a Firefox browser extension that displays your Bookmarks Menu in a clean, vertical layout. It supports multiple tiers of folders and makes it easy to browse and open your bookmarks.

## Features

- Reads bookmarks from your Firefox Bookmarks Menu
- Supports multi-level folders and nested bookmarks
- Simple, vertical UI for easy navigation
- Click to open bookmarks in a new tab

## Installation

1. Open Firefox and go to `about:debugging#/runtime/this-firefox`.
2. Click "Load Temporary Add-on..." and select the `manifest.json` file.

## Usage

1. Click the Bookie extension icon in your browser toolbar.
2. Browse your bookmarks in a vertical, folder-based layout.
3. Click any bookmark to open it in a new tab.

## Development

No external dependencies required.

Click 'Reload' to test changes and 'Inspect' to debug the running app.

## Sidebar / Vertical Tab Support

- Bookie includes a sidebar panel so it can be used with Firefox's vertical tabs or sidebar mode.
- To open Bookie in the sidebar: open `about:addons`, find the Bookie extension, and click the sidebar icon (or use the extension menu).
- When loaded as a sidebar, Bookie displays the same Bookmarks Menu view optimized for a tall vertical layout.
